# Weather App

A simple and responsive weather forecasting application.

## Features
- Current weather information
- Temperature and humidity data
- Location-based forecasts
- Clean user interface

## Tech Stack
- HTML
- CSS
- JavaScript
- Weather API

## Setup

1. Clone the repository
2. Add your API key
3. Run the application

## Future Improvements
- Weekly forecasts
- Weather maps
- Severe weather alerts
